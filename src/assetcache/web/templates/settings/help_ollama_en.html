{# Ollama (local) — free + step-by-step setup #}

Free · runs on your PC

  1. Install Ollama (Windows / Mac / Linux supported)
  2. Open a terminal: ollama pull gemma4:e4b (≈ 9.6 GB download)
  3. Verify: ollama list → gemma4:e4b should be listed

The model ID in the input above must match gemma4:e4b (the default).
